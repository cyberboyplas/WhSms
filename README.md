# - WhSms -

Enviar sms de forma anónima, con texto personalizado

# - Instalación -

`pkg install git` 

`git clone https://github.com/WhBeatZ/WhSms.git`

`pkg install bash`

`cd WhSms`

`chmod +x *`

`bash whsms.sh`
